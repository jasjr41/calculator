# Calculator Application

A simple calculator application built with Python's Tkinter library. This calculator performs basic arithmetic operations with a clean, intuitive user interface.

## Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **User-Friendly Interface**: Simple layout with clearly labeled buttons
- **Real-time Display**: Shows input and results in a clear entry field
- **Error Handling**: Built-in handling for basic calculation errors
- **Clear Function**: Reset the display with a single click

## Operations Supported

1. **Addition** (+)
2. **Subtraction** (-)
3. **Multiplication** (*)
4. **Division** (/)

## How to Use

1. Enter the first number by clicking the number buttons
2. Select an operation (+, -, *, /)
3. Enter the second number
4. Press the "=" button to see the result
5. Use "Clear" to reset the calculator
